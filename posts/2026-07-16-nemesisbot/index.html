2026-07-16
#agent  #workflow  #document-summary 

起因很简单：粉丝抽奖，一位中奖的老哥，他不要别的奖品，提了一个需求——让我给他写一个工具，能帮他批量分析一堆文档，然后做总结归纳，最后写一份摘要。

需求拆开看，其实就是一条流水线：

  • 把一批文档喂进去；
  • 逐个分析、提取要点；
  • 把要点归并、去重；
  • 最后生成一份总结摘要。

这种「读一堆东西 → 归纳 → 出结论」的活儿，恰好是 Agent 最擅长的事。所以我没从头造轮子，直接基于 NemesisBot 搭了一个 Agent 工作流：一个 Agent 负责遍历和拆解文档，中间环节做归并，最后一个 Agent 收口写摘要。对使用者是黑盒——把文档丢进工作目录，剩下的交给 Agent 跑就行。

关于 NemesisBot#

NemesisBot 不只是为了这一个需求做的——它是一个用 Rust 写的本地 AI Agent 平台，带 8 层安全审计体系、工作流引擎、分布式集群，Windows / Linux / macOS / Android 都能跑。这次抽奖需求，只是它一个很典型的应用场景：把「反复读文档 + 总结」这种费时费力的活儿，丢给 Agent 自动跑。

中奖的老哥，工具给你弄好了，enjoy。
